내 라인, 내가 할 수 있는 챔피언, 금지 챔피언을 반영해서 추천 점수와 이유를 보여줍니다. 아직 실제 밴픽 API 연동 전 단계라서 현재는 로컬 시뮬레이터 버전입니다.

1. 라인 선택

먼저 내가 플레이할 라인을 선택하세요. 이후 해당 라인에 맞는 챔피언 풀만 보여줍니다.

2. 내 챔피언 풀 선택

클릭하면 챔프풀에 추가됩니다. 다시 클릭하면 해제됩니다. 최소 2개 이상 고르면 추천 품질이 더 좋아집니다.

선택한 챔프풀

한 챔피언만 선택하면 추천 결과가 단조로워질 수 있습니다. 탱커/캐리/안전픽을 섞으면 더 현실적인 추천이 나옵니다.

3. 밴 고려

현재 게임에서 못 쓰는 챔피언을 골라 제외하세요. 내 챔프풀에 있는 챔피언도 밴 처리하면 추천에서 빠집니다.
클릭 = 밴/해제

현재 밴 목록
